# Lançamentos: Monitor de Rifas 🎟️

Bem-vindo ao repositório oficial de distribuições do **Monitor de Rifas - @minisdaana**.

## 📥 Como Instalar
1. Vá até a seção [Releases](https://github.com/pHgon/raffle-monitor-releases/releases).
2. Baixe o arquivo mais recente: `Monitor_de_Rifas_Setup.exe`.
3. Execute o instalador no seu Windows.
4. O app será aberto automaticamente após a instalação.

## 🔄 Atualizações Automáticas
Uma vez instalado, o aplicativo verificará se existem novas versões sempre que for iniciado. Se houver uma atualização disponível, você poderá baixar e instalar com apenas um clique diretamente de dentro do app.

## 🧪 Programa Beta
Quer testar as correções e novidades antes de todo mundo?
1. Abra o aplicativo.
2. Clique **5 vezes** sobre o título **"Monitor de Rifas"** no topo da tela.
3. Você receberá um aviso confirmando a entrada no programa de testes.
4. Para sair, basta repetir o processo.
